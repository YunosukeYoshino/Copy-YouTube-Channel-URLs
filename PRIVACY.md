# プライバシーポリシー

最終更新: 2025-11-17

拡張機能: Copy YouTube Channel URLs
リポジトリ: https://github.com/YunosukeYoshino/Copy-YouTube-Channel-URLs

このプライバシーポリシーは、当拡張機能がどのようなデータを収集し、どのように扱うかを説明するものです。

## 1. 要約
本拡張機能は、ユーザーが拡張のボタンをクリックした際に、現在表示中の YouTube ページから可視の動画サムネイルに含まれる再生用 URL（watch URL）とタイトルを取得し、ユーザーのクリップボードへコピーするために利用します。取得したデータは拡張内およびブラウザのローカルストレージにのみ保存され、外部のサーバーへ送信することはありません。

## 2. 収集する情報
拡張機能は次の情報をユーザー操作に応じて収集します（自動で外部に送信されることはありません）：

- YouTube ページ上の可視動画から抽出した情報
  - 動画の再生 URL（例: https://www.youtube.com/watch?v=...）
  - 動画タイトル（可能な場合）
- コピー操作に関する簡易ログ（ローカル保存）
  - コピーした URL の個数、操作の成否などの短期的なデバッグ情報

注意: 拡張はユーザーがボタンを押したときなど明示的なユーザー操作があった場合にのみページの DOM を読み取ります。

## 3. 収集目的
収集した情報は次の目的のみに使用します：

- ユーザーが選択した動画 URL をクリップボードにコピーするため
- 動作に関する簡易ログをローカルに残し、デバッグや動作検証を行うため（長期保管は行いません）

## 4. 外部送信・第三者共有
当拡張機能は、収集したデータを外部のサーバーや第三者に送信しません。すべての処理はユーザーのブラウザ内で行われます。

## 5. 保存期間
- クリップボードにコピーした URL 自体はユーザーのクリップボード上に残ります（ブラウザ／OS のクリップボード管理に依存します）。
- 拡張内部で保持する簡易ログ（`chrome.storage.local` 等）は、デバッグ目的で短期間のみ保持されます。ログの保持が不要な場合は、拡張をアンインストールすることで削除されます。

## 6. 権限と理由
拡張が要求する主な権限とその理由は以下の通りです：

- host permissions: `https://www.youtube.com/*`
  - YouTube ページ上の可視動画情報（URL・タイトル）を抽出するために必要です。
- `activeTab` / `tabs`（実装により片方のみ）
  - ポップアップからアクティブタブへメッセージを送り、コンテンツスクリプトと連携して動画を収集するために使用します。
- `clipboardWrite`（manifest に記載されている場合）
  - ユーザーが選択した URL をクリップボードに書き込むために使用します。navigator.clipboard API を利用するため、ユーザー操作がある場合は明示的な権限が不要なこともありますが、ストア申請の説明のために記載しています。

不要な権限がある場合は削除しており、権限の最小化に努めています。

## 7. サードパーティサービス
現状、外部の分析やログ送信といった第三者サービスは利用していません。

## 8. ユーザーの権利
この拡張が収集するデータは基本的にユーザーのブラウザ内に留まるため、削除や閲覧はローカルで行ってください。拡張をアンインストールするとローカルストレージに保存されたデータも削除されます。
